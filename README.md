# HandyUtils (핸디 유틸) 🛠️

**HandyUtils**는 개발자에게 유용한 순수 자바스크립트(Vanilla JS) 기반의 경량 도구들을 모아둔 저장소입니다.
복잡한 설정 없이(Zero Configuration), 누구나 바로 가져다 쓸 수 있는 실용적인 유틸리티를 지향합니다.

## � 포함된 도구 목록 (Tools)

### 1. [LiteGantt](./lite-gantt)
초경량 순수 자바스크립트 간트 차트 라이브러리입니다.
- **특징**: 의존성 없음, 자동 무지개 색상, 반응형 디자인.
- **용도**: 경력 기술서, 프로젝트 일정, 로드맵 시각화.
- [자세히 보기](./lite-gantt/README.md)
